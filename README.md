## 🎯 What This Project Does

This project simulates a **Railway Traffic Control System** that manages train movements, signal states, and scheduling logic. It models real-world railway operations including signal control, collision avoidance, and route management.

---

## ✨ Features

- **Signal State Management** — Controls green/yellow/red signal transitions
- **Train Scheduling** — Manages departure/arrival times and track allocation
- **Collision Avoidance Logic** — Detects and prevents conflicting train paths
- **Route Management** — Assigns optimal tracks based on current traffic
- **Simulation Mode** — Run controlled test scenarios for validation

---

## 🛠️ Tech Stack

- **Language:** Python 3.10+
- **Paradigm:** Object-Oriented Simulation
- **Configuration:** Environment variables via `.env`

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/ujjawalranjan09/project-3.git
cd project-3

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env

# Run the simulation
python railway-traffic-control-system/main.py
```
